# Loading Animation Project

A responsive loading animation that dissolves into a "Coming Soon" page. Optimized for all device types.

## Features

- Responsive loading animation (desktop and mobile versions)
- Smooth dissolve transition
- Device-optimized video playback
- Session-based animation display (shows once per session)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
